**Objectif** : Apprendre à se connecter et naviguer

```
"Bonjour ! Dans cette vidéo, nous allons apprendre à nous connecter
à KA Farm et à naviguer entre les différentes sections du site."

"Au sec, il est 8h du matin sur le terrain de Niayes,
nous allons tout vous montrer étape par étape."
```

**(0:30-1:00) Connexion**

```
"Première étape : la connexion. Vous avez reçu vos identifiants
par SMS ou WhatsApp : votre email et votre mot de passe."

"Allez sur l'adresse fournie par votre admin, cliquez sur
'Se connecter' et entrez vos identifiants."

"Une fois connecté, vous voyez le tableau de bord avec toutes
les sections disponibles."
```

**(1:00-1:45) Navigation**

```
"Maintenant, voici comment naviguer. Sur ordinateur, utilisez
le menu à gauche (la sidebar)."

"Sur téléphone, cliquez sur le menu hamburger (les trois lignes)
en haut à gauche."

"Les sections disponibles sont : Accueil, Tableau de Bord, Cultures,
Stocks, Finances, et bien plus encore."
```

**(1:45-2:30) Section par section**

```
"Voyons rapidement chaque section :"

- 🏠 Accueil : Vue d'ensemble
- 📊 Tableau de Bord : Toutes les statistiques importantes
- 🌾 Cultures : Suivit des cultures
- 📦 Stocks : Inventaire et alertes
- 💰 Finances : Revenus, dépenses, solde
- 👥 Employés : Personnel et pointage
- 💬 Discussions : Messagerie interne
```

**(2:30-3:00) Conclusion**

```
"Maintenant, vous savez vous connecter et naviguer.
Dans la prochaine vidéo, nous allons apprendre à gérer les cultures."

"À bientôt et bon travail sur le terrain !"
```

## 📹 Tutoriel 2 : Gestion des Cultures

**Durée** : 5 minutes  
**Objectif** : Apprendre à ajouter et gérer une culture

### Script

**(0:00-0:30) Introduction**

```
"Bonjour ! Aujourd'hui, nous allons apprendre à gérer les cultures
dans KA Farm. C'est une étape cruciale pour suivre l'évolution
de vos plantations."
```

**(0:30-1:15) Ajouter une culture**

```
"Allons sur la section Cultures en cliquant sur l'icône 🌾."

"Pour ajouter une nouvelle culture, cliquez sur le bouton
'Ajouter une Culture' en haut à droite."

"Remplissez le formulaire :"

"1. ID Culture : C-006 (suivez la numérotation existante)"
"2. Nom : Tomate Mongal F1 (nom complet de la variété)"
"3. Parcelle : Parcelle Nord - Planche 2"
"4. Date Semis : 2026-06-01 (date de plantation)"
"5. Date Récolte : 2026-08-15 (date prévue)"
"6. Statut : Croissance (état actuel)"
"7. Eau : Optimale (besoin en irrigation)"
"8. Engrais : OK (état de fertilisation)"

"Cliquez sur Enregistrer. La culture apparaît maintenant dans la liste."
```

**(1:15-2:30) Comprendre les statuts**

```
"Les statuts des cultures sont importants :"

"🟡 Semis : La culture vient d'être plantée"
"🟢 Croissance : La culture se développe normalement"
"🟣 Floraison : La culture est en fleur"
"🟠 Récoltable : Prête à être récoltée"
"⚪ Récolté : Déjà récoltée"

"Changez le statut chaque semaine pour suivre l'évolution."
```

**(2:30-3:45) Modifier une culture**

```
"Pour modifier une culture existante, trouvez-la dans la liste
et cliquez sur l'icône ✏️ (modifier)."

"Vous pouvez changer n'importe quel champ. Par exemple,
si vous avez arrosé, mettez 'Eau' sur 'Optimale'."

"N'oubliez pas de cliquer sur Enregistrer après modification."
```

**(3:45-4:30) Bonnes pratiques**

```
"Quelques règles d'or :"

"✅ Enregistrez la culture le jour du semis"
"✅ Mettez à jour le statut chaque semaine"
"✅ Ajoutez des notes dans le champ 'notes'"
"✅ Utilisez des noms complets, pas d'abréviations"

"❌ Évitez les doublons"
"❌ N'oubliez pas de changer le statut"
```

**(4:30-5:00) Exercice et conclusion**

```
"Exercice : Ajoutez une culture de choux avec les informations
suivantes : C-007, Chou Cabus, Parcelle Sud, 2026-07-01, 2026-09-30,
Croissance."

"Dans la prochaine vidéo, nous allons apprendre à gérer les stocks."

"À bientôt !"
```

## 📹 Tutoriel 3 : Gestion des Stocks

**Durée** : 4 minutes  
**Objectif** : Apprendre à gérer l'inventaire

### Script

**(0:00-0:30) Introduction**

```
"Bonjour ! Dans cette vidéo, nous allons apprendre à gérer
les stocks dans KA Farm. C'est essentiel pour éviter les
pénuries et maîtriser vos coûts."
```

**(0:30-1:30) Ajouter un stock**

```
"Allons sur Stocks en cliquant sur l'icône 📦."

"Cliquez sur 'Ajouter un Stock'. Voici les champs à remplir :"

"ID : S-006"
"Nom : Engrais NPK 15-15-15"
"Catégorie : Engrais"
"Quantité : 50 (quantité actuelle)"
"Max : 200 (quantité maximale souhaitée)"
"Unité : sacs"

"Cliquez sur Enregistrer. Le stock apparaît dans la liste."
```

**(1:30-2:30) Comprendre les alertes**

```
"Regardez cette icône ⚠️. Elle signifie 'stock bas'."

"Une alerte s'affiche automatiquement quand :"
"Quantité ≤ 20% de Quantité Maximale"

"Exemple : Si Max = 100, alerte si Quantité ≤ 20"

"Quand vous voyez cette alerte, commandez du stock."
```

**(2:30-3:15) Mettre à jour un stock**

```
"Pour mettre à jour un stock, cliquez sur ✏️ (modifier)."

"Si vous avez utilisé 10 sacs d'engrais, changez la quantité
de 50 à 40."

"Si vous avez reçu une livraison de 30 sacs, changez de 40 à 70."

"Cliquez sur Enregistrer."
```

**(3:15-4:00) Exercice et conclusion**

```
"Exercice : Ajoutez un stock de semences de piment : S-007,
Semences Piment Oiseau, Catégorie Semences, Quantité 15,
Max 30, Unité sachets."

"Dans la prochaine vidéo, nous allons apprendre à gérer
les finances."

"À bientôt !"
```

## 📹 Tutoriel 4 : Gestion Financière

**Durée** : 6 minutes  
**Objectif** : Maîtriser l'enregistrement des transactions

### Script

**(0:00-0:45) Introduction**

```
"Bonjour ! La gestion financière est cruciale pour savoir
où va l'argent de l'exploitation. Dans cette vidéo, nous allons
apprendre à enregistrer les revenus et les dépenses."
```

**(0:45-2:00) Enregistrer une vente (Revenu)**

```
"Allons sur Finances en cliquant sur 💰."

"Cliquez sur 'Ajouter Transaction'. Sélectionnez le type 'Revenu'."

"Remplissez le formulaire :"

"ID : F-013"
"Description : Vente 100 kg tomates"
"Catégorie : Vente Légumes"
"Montant : 65000 (en FCFA)"
"Date : 2026-11-16"

"Cliquez sur Enregistrer. Le revenu est ajouté."
```

**(2:00-3:15) Enregistrer un achat (Dépense)**

```
"Maintenant, enregistrons une dépense."

"Même processus, mais type = 'Dépense'."

"ID : F-014"
"Description : Achat 10 sacs engrais"
"Catégorie : Engrais"
"Montant : 50000"
"Date : 2026-11-15"

"La dépense est enregistrée."
```

**(3:15-4:30) Comprendre les statistiques**

```
"En haut de la page, vous voyez 3 indicateurs clés :"

"💵 Total Revenus : Somme de tous les revenus"
"💸 Total Dépenses : Somme de toutes les dépenses"
"💰 Solde : Revenus - Dépenses"

"Exemple : Si Revenus = 2 000 000 et Dépenses = 1 500 000,
alors Solde = +500 000 F (positif ✅)"
```

**(4:30-5:30) Catégories importantes**

```
"Voici les catégories les plus utilisées :"

"Revenus :"
"- Vente Légumes"
"- Vente Fruits"
"- Vente Animaux"

"Dépenses :"
"- Semences"
"- Engrais"
"- Carburant"
"- Salaires"
```

**(5:30-6:00) Exercice et conclusion**

```
"Exercice : Enregistrez une vente de 200 kg d'oignons à 500 F/kg
(Type : Revenu, Description : Vente oignons, Montant : 100000)."

"Dans la prochaine vidéo : la gestion du personnel."

"À bientôt !"
```

## 🛠️ Matériel de Tournage

### Équipement Minimal

- 📱 **Smartphone** : iPhone ou Android (caméra HD minimum)
- 🎤 **Micro** : Micro cravate ou micro externe (optionnel mais recommandé)
- 📱 **Support** : Trépied téléphone ou livre posé sur table
- 💡 **Éclairage** : Lumière naturelle (près d'une fenêtre)

### Logiciels de Montage (Gratuits)

1. **CapCut** (mobile/desktop) - Très facile à utiliser
2. **DaVinci Resolve** (desktop) - Professionnel et gratuit
3. **OpenShot** (desktop) - Open source et simple

### Configuration Recommandée

```
Résolution : 1080p (1920x1080)
Format : MP4
Durée : 3-6 minutes par vidéo
Taille fichier : < 100 MB par vidéo
Cadre : Paysage (16:9) pour desktop, Portrait (9:16) pour mobile
```

## 📋 Comment Former l'Équipe ?

### Option 1 : Formation Présentielle (Recommandé)

**Durée** : 2-3 heures  
**Matériel** : Projecteur + ordinateur

**Déroulé** :

1. **Module 1** (30 min) : Découverte de KA Farm
2. **Module 2** (25 min) : Gestion des Cultures
3. **Module 3** (20 min) : Gestion des Stocks
4. **Pause** (10 min)
5. **Module 4** (30 min) : Gestion Financière
6. **Module 5** (25 min) : Gestion du Personnel
7. **Module 6** (20 min) : Astuces et Bonnes Pratiques
8. **Quiz** (10 min) : Évaluation
9. **Exercices pratiques** (20 min)

### Option 2 : Formation en Autonomie

**Durée** : 3-5 jours  
**Matériel** : Accès internet + ordinateur/smartphone

**Déroulé** :

- Jour 1 : Lire TEAM_TRAINING_GUIDE.md (Modules 1-3)
- Jour 2 : Regarder les tutoriels vidéo + pratique
- Jour 3 : Lire Modules 4-6 + exercices
- Jour 4 : Quiz + validation compétences
- Jour 5 : Utilisation autonome + support

### Option 3 : Formation Mixte

**Durée** : 2 jours

**Jour 1** : Formation présentielle (Modules 1-3)
**Jour 2** : Formation présentielle (Modules 4-6) + exercices pratiques

## ✅ Évaluation de la Formation

### Quiz de Validation

Le quiz est inclus dans **TEAM_TRAINING_GUIDE.md** (Section "Évaluation de la Formation").

**4 questions** :

1. Nombre de sections dans KA Farm
2. Quand une alerte de stock bas apparaît
3. Comment enregistrer une vente
4. Que faire si la session expire

**Score minimum** : 10/12 avec la checklist de compétences

### Checklist de Compétences

Cocher chaque élément maîtrisé :

- [ ] Se connecter à son espace
- [ ] Naviguer entre les sections
- [ ] Ajouter une culture
- [ ] Modifier le statut d'une culture
- [ ] Ajouter un article en stock
- [ ] Interpréter les alertes de stock
- [ ] Enregistrer une transaction financière
- [ ] Lire les statistiques financières
- [ ] Ajouter un employé
- [ ] Pointer les présences
- [ ] Enregistrer un paiement
- [ ] Contacter le support

**Validation** :

- 10-12 : ✅ Excellent, prêt à utiliser KA Farm
- 7-9 : ✅ Bon, quelques révisions recommandées
- < 7 : ⚠️ Formation complémentaire nécessaire

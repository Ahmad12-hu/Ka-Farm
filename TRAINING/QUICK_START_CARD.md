## ⚡ Actions les Plus Courantes

### 🌾 Ajouter une Culture

1. Aller sur **Cultures** 🌾
2. Cliquer **"Ajouter une Culture"**
3. Remplir : ID, Nom, Parcelle, Dates, Statut
4. **Enregistrer**

### 📦 Ajouter un Stock

1. Aller sur **Stocks** 📦
2. Cliquer **"Ajouter un Stock"**
3. Remplir : ID, Nom, Catégorie, Quantité, Max, Unité
4. **Enregistrer**

### 💰 Enregistrer une Vente

1. Aller sur **Finances** 💰
2. Cliquer **"Ajouter Transaction"**
3. Type : `Revenu`
4. Remplir : Description, Catégorie, Montant, Date
5. **Enregistrer**

### 💸 Enregistrer un Achat

1. Aller sur **Finances** 💰
2. Cliquer **"Ajouter Transaction"**
3. Type : `Dépense`
4. Remplir : Description, Catégorie, Montant, Date
5. **Enregistrer**

### 👤 Ajouter un Employé

1. Aller sur **Employés** 👥
2. Cliquer **"Ajouter Employé"**
3. Remplir : ID, Nom, Téléphone, Rôle, Taux, Statut
4. **Enregistrer**

### ✅ Pointer une Présence

1. Aller sur **Employés** 👥
2. Trouver l'employé
3. Cliquer **📋 (pointage)**
4. Date + Statut (Présent/Absent/Demi-journée)
5. **Enregistrer**

## 🚨 Alertes Importantes

### ⚠️ Stock Bas

```
Quantité ≤ 20% du Maximum
→ Action : Commander du stock
```

### ⚠️ Session Expirée

```
"Accès refusé"
→ Action : Se déconnecter et se reconnecter
```

### ⚠️ Erreur de Sauvegarde

```
"Données non sauvegardées"
→ Action : Vérifier internet + Actualiser (F5)
```

## 📝 Règles d'Or

### ✅ À Faire

- ✍️ Enregistrer les données **le jour même**
- ✅ Vérifier avant de cliquer sur **Enregistrer**
- 📝 Ajouter des **notes** pour les cas particuliers
- 🔒 Se **déconnecter** après utilisation
- 📱 Mettre à jour les stocks **après chaque utilisation**

### ❌ À Éviter

- ❌ Partager son mot de passe
- ❌ Laisser la session ouverte
- ❌ Supprimer sans confirmation
- ❌ Créer des doublons
- ❌ Oublier de changer les statuts

## 🆘 En Cas de Problème

### 1. Problème de connexion

```
→ Se déconnecter
→ Vider le cache (Ctrl + Shift + Delete)
→ Se reconnecter
```

### 2. Données perdues

```
→ Actualiser la page (F5)
→ Vérifier la connexion internet
→ Contacter l'admin si nécessaire
```

### 3. Erreur d'affichage

```
→ Actualiser la page (F5)
→ Vider le cache
→ Réessayer
```

### 4. Contact Support

```
📧 Email : admin@kafarm.sn
📱 Téléphone : [Numéro admin]
💬 Discussion : Section "Discussions"
```

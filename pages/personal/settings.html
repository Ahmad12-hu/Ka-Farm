Configuration de l'Assistant IA

Pour activer l'assistant IA de KA Farm, ajoutez votre clé API Google Gemini. Obtenir une clé API gratuite

📖 Guide rapide : Comment obtenir ma clé API ?

1. Va sur aistudio.google.com/app/apikey

2. Connecte-toi avec ton compte Google

3. Clique sur "Get API Key"

4. Crée un projet ou sélectionne un projet existant

5. Copie la clé (format: AIzaSy...)

6. Colle-la ci-dessus et clique sur "Enregistrer"

💡 Astuce : La clé est gratuite et limitée à ~60 requêtes/minute. Parfait pour KA Farm !

🔒 Stockée localement dans votre navigateur. Jamais partagée.

⚠️ Note : L'assistant IA utilise l'API Google Gemini qui est gratuite dans sa limite d'utilisation. Au-delà, des frais peuvent s'appliquer. Consultez la tarification.
